Коротко про мову C Sharp

Мова C Sharp (C#) – сучасна крос-платформова об'єктно-орієнтована мова програмування. Розробку мови як відповідь на мову Java почали фахівці Microsoft під керівництвом Андерса Хейлсберга у 2000 році. Одночасно з мовою C#, почалася розробка середовища виконання для цієї мови, що отримало назву .NET Framework.

Починаючи з першої версії (2001р.), мова має такі конструкції:

  • Умовні конструкції, цикли з лічильником та умовою
  • Cтворення та перехоплення виключень
  • Класи, структури, перераховні типи, інтерфейси
  • Обчислювані поля класів (computed properties)
  • Делегати (як покажчики на функцію в мові С++, але більш безпечні)
  • Прямокутні та "зшиті" багатовимірні масиви

Наступна версія (2002р.) відзначилася появою:

  • Відкладеної типізації (Generics)
  • Лямбда-вирази
  • Передача параметрів функцій за посиланням (ref, in, out)

Відкладена типізація дозволила створювати нові структури даних та більш ефективно розробляти алгоритми для обробки даних.

На шляху до крос-платформності

C# одразу показав себе як чудова мова програмування, але від самого початку реалізація цієї мови (.NET Framework) була пропрієтарною та призначеною лише для ОС Windows. Windows була найбільш популярною операційною системою тих часів, але почала втрачати долю ринку з розповсюдженням MacOS та Linux. Тому розробники на C# вже не могли розробляти програми "для всіх" і мали задовольнятися специфічною нішею розробки для Windows. На C# можна було писати консольні програми, модулі, графічні програми, web-додатки, ігри – але це все працювало лише на Windows.

Першим рішенням стала неофіційна реалізація "Mono" – розроблена в 2004 групою програмістів під керівництвом Мігеля де Ікаса. Це середовище виконання дозволило об'єднати "під одним дахом" Windows, Linux та MacOS.

Однак розвиток мови C# та невід'ємної від неї платформи .NET Framework відбувався досить повільно. Окрім Linq, не було майже ніяких інновацій, а Mono просто "списували" з офіційної версії.

Кардинальний переворот відбувся у 2014 році. Тоді розпочалася розробка .NET Core – офіційної крос-платформової реалізації .NET з відкритим кодом. Будь-хто міг долучитися до розробки. Був розроблений новий компілятор Roslyn, більш гнучкий, оптимізований та легко розширюваний. До нових версій мови C# були додані засоби асинхронного програмування, кортежі, виділення пам'яті для непримітивних об'єктів на стеку, інтерполяція текстових рядків, деконструкція структур, switch-конструкція з кортежами і т.д.

Також був з нуля розроблений фреймфорк ASP.NET Core – сучасний фреймворк для веб-додатків, що може відзначитися швидкістю, гнучкістю, крос-платформовістю та доступом до всіх бібліотек C#.
